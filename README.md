Average Alexa Skills Certification Times
===

Deployed on AWS
- 3 Lambda functions
- 2 DynamoDB tables
- 1 S3 bucket
- 1 CloudFront distribution


HTML pages are template based and rendered with [mustache.js](https://github.com/janl/mustache.js/) library.


Development
===

Initial setup

```
nvm use
yarn install
```

Run tests

```
grunt test
```
